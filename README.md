# RunningCat — GitHub Pages Deployment

This is a browser/WebGL deployment port of the original Python PyOpenGL project.

## Play

After GitHub Pages deploys successfully, open:

https://shouravsaha573.github.io/Cat-Runner-Game-Deploy/

## Controls

- `Space`: start / restart
- `A` or Left Arrow: move left
- `D` or Right Arrow: move right
- `W` or Up Arrow: jump
- `S` or Down Arrow: slide

## Deployment

This repository includes a GitHub Actions workflow at `.github/workflows/pages.yml`.
In repository **Settings → Pages**, set **Source** to **GitHub Actions**.
