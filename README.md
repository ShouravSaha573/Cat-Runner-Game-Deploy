# RunningCat

This is a browser/WebGL deployment port of the original Python PyOpenGL project.

## Play
https://shouravsaha573.github.io/Cat-Runner-Game-Deploy/

## Controls

- `Space`: start / restart
- `A` or Left Arrow: move left
- `D` or Right Arrow: move right
- `W` or Up Arrow: jump
- `S` or Down Arrow: slide
